# expensive-meetings
A minimal web-app for calculating just how expensive meetings really are

## To Use:
1. Go to [this](http://streetlight.github.io/expensive-meetings/) link.
2. Enjoy

## To Run:
1. Pull down
2. Serve index.html with any engine you like
3. Enjoy
